# Large Text Toggle for GNOME

A simple GNOME Shell extension to toggle the "Large Text" accessibility setting (1.25 scaling) with a single click from the top panel.

## Installation

1. Clone this repo into your extensions directory:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/largetext-extension.git](https://github.com/YOUR_USERNAME/largetext-extension.git) ~/.local/share/gnome-shell/extensions/largetext@yourname.dev
